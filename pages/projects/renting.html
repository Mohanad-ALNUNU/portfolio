RENTING CARS (BackEnd)

Developed a Java Spring Boot project using the microservices architecture pattern, with inter-microservice communication via Kafka and data storage in MongoDB.

Created a responsive user interface using Angular, ensuring compatibility and a seamless user experience across all devices.

Integrated Bootstrap components in the UI design to enhance its visual appeal and user-friendliness.

Ensured adherence to clean code principles, best practices, and conventions throughout the project.

Implemented comprehensive unit tests using JUnit 5 and Mockito to ensure reliable functionality.

Utilized Swagger for documenting and visualizing the project's endpoints.

RENTING CARS (FrontEnd)

Developed a responsive user interface using Angular, leveraging APIs from the multi-module BackEnd project to provide users with a seamless experience for renting vehicles.

Ensured compatibility across various devices, ensuring that users can easily access and use the application on different platforms.

Collaborated closely with the BackEnd team to align the FrontEnd functionality with the microservices architecture.

Integrated Bootstrap components to enhance the visual appeal and user-friendliness of the interface.
