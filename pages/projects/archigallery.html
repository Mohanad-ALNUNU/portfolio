2022 - Architectural Gallery

Created an immersive 3D architectural website that provides users with a user-friendly platform to explore intricate building designs and urban plans.

The website features three distinct modules, each showcasing unique architectural projects and an urban plan, all carefully designed to highlight the intricate details of the structures.

The live site for the project can be accessed at: Click to access to live project

The project leverages a technical environment including THREE-JS, SketchUp, Blender, and Photoshop to deliver a visually stunning and interactive experience.

The user-friendly website was developed using Three.js, with the 3D architectural modules meticulously crafted and refined in Blender and SketchUp.

To ensure a seamless user experience, a smooth navigation system was implemented, allowing users to explore the modules effortlessly using next and previous buttons.

The user experience was further enhanced by integrating three control modes: Orbit mode (default), FPS mode, and Fly mode, offering users multiple ways to interact with the architectural designs.

An information window was implemented to provide precise details about the buildings. Users can click on a building to access comprehensive information about its structure and design.

The project successfully features three modules:

  • Module 1: Consisting of four distinct buildings.
  • Module 2: Showcasing a unique and specific building.
  • Module 3: Created as an urban plan, highlighting a city composed of numerous buildings.
